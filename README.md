# Xcode CodeSnippets

![Xcode Snippet View](snipped-view.png)


To install these snippets open your terminal and execute:

```
cd ~/Library/Developer/Xcode/UserData
git clone https://github.com/phranck/Xcode-CodeSnippets.git CodeSnippets
```
